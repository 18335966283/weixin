# 大家来找茬腾讯版

待完善

## 功能
- [x] 识别不同点，标注红色
- [x] 打开标注后的图片
- [ ] 自动点击手机屏幕

## 运行
- 安装 Node.js 8.x 以上
- 安装 adb 工具
- 安卓手机连接数据线
- 打开 USB 调试
- 安装依赖 `npm i`
- 运行脚本 `npm start`
