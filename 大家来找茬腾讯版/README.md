# 大家来找茬腾讯版

## 功能
- [x] 识别不同点，并标注颜色
- [x] 显示标注过的图片
- [ ] 自动点击手机屏幕（时间关系，暂时搁置）

## 要求
- Node.js 8.x 以上
- 安卓手机连接数据线，并打开 USB 调试

## 运行
```bash
npm i
npm start
```
